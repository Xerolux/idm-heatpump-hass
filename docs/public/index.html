IDM Heatpump

Home Assistant Integration – 100% lokal via Modbus TCP

HA 2025.12.0+ HACS MIT License Modbus TCP

Features

🌡️ System Monitoring

100+ sensors for temperatures, pressures, flow rates, energy, and runtime hours

🔧 Heating Circuits A–G

Up to 7 circuits with individual setpoints and mode control

🏠 Zone Modules

Up to 10 zones with 8 rooms each (room thermostat function)

☀️ Solar & PV

Solar hot water, PV surplus utilization, battery monitoring

💧 Domestic Hot Water

DHW setpoint and priority control

⚡ Full Control

Write setpoints, change modes, acknowledge errors via HA UI

🔑 Installer Codes

Optional sensors for Fachmann Ebene 1 & 2 codes – refreshed every minute

🛡️ EEPROM Protection

88 write-protected registers, exponential backoff on connection errors

Quick Start

  1. Click the HACS button above — or manually add Xerolux/idm-heatpump-hass as a custom repository
  2. Download "IDM Heatpump" and restart Home Assistant
  3. Go to Settings → Devices & Services → Add Integration → IDM Heatpump
  4. Enter your heat pump IP address and Modbus port (default: 502)
  5. Select heating circuits, zones, and optional features → Done!
