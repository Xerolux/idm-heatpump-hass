IDM Heatpump

Home Assistant Integration via Modbus TCP

HA 2025.8.0+ HACS MIT License Modbus TCP

Features

System Monitoring

100+ sensors for temperatures, pressures, flow rates, energy, runtime hours

Heating Circuits A-G

Up to 7 circuits with individual setpoints and mode control

Zone Modules

Up to 10 zones with 8 rooms each (room thermostat function)

Solar & PV

Solar hot water, PV surplus utilization, battery monitoring

Domestic Hot Water

DHW setpoint and priority control

Full Control

Write setpoints, change modes, acknowledge errors via HA UI

Installer Codes

Optional sensors showing the current Installer Level 1 & 2 access codes, refreshed every minute

Quick Start

  1. Add custom repository in HACS: https://github.com/Xerolux/idm-heatpump-hass
  2. Download "IDM Heatpump" and restart HA
  3. Go to Settings → Devices & Services → Add Integration
  4. Enter heat pump IP address and Modbus port (default: 502)
  5. Select heating circuits and zones → Done!
